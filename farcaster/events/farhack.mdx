## Prizes

<AccordionGroup>
  <Accordion title="🖼️ Best Use of Pinata in a Farcaster Frame - $500">
    This prize will be awarded to the team who builds the most interesting Farcaster Frame that uses Pinata.
    <Info>
      **Qualification Requirements**
      1. Build a Farcaster Frame
      2. Use either the Farcaster API, Farcaster Auth, or IPFS from Pinata
    </Info>
  </Accordion>

  <Accordion title="🔨 Best Use of Pinata in a Cast Action - $500">
    This prize will be awarded to the team who builds the best use case of Pinata in a Cast Action.
    <Info>
      **Qualification Requirements**
      1. Build a Farcaster Cast Action
      2. Use either the Farcaster API, Farcaster Auth, or IPFS from Pinata
    </Info>

  </Accordion>

  <Accordion title="📱 Best Channel Specific Client using Pinata Farcaster API - $500">
    This prize will be awarded to the team who builds the best channel specific client using the Farcaster API.
    <Info>
      **Qualification Requirements**
      1. Build a Farcaster client for a specific channel
      2. Use Pinata's Farcaster API (or FDK read and write methods)
    </Info>

  </Accordion>
</AccordionGroup>
